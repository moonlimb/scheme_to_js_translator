#(random a) --> returns a random number between 0 and a-1
# ex. (random 100) --> returns a random number b/w 0 and 99
#	(+ 1 (random (100))) --> returns a random number b/w 1 and 100


Math.floor((Math.random()*100)+1);
--> returns a random number b/w 1 and 100

Math.random();
--> returns a number from 0 to 1
